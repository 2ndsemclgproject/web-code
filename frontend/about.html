Project Overview

BreathIQ is a comprehensive, multi-disciplinary IoT environmental and respiratory monitoring system. It was developed to bridge the gap between complex hardware sensor networks and modern, accessible medical data visualization.

The system actively measures Carbon Dioxide (CO2), Volatile Organic Compounds (VOCs), Oxygen (O2), Temperature, and Humidity. By applying a custom algorithm to these isolated metrics, the system calculates a real-time Breath Quality Index (BQI), providing instant risk assessments and health recommendations.

System Architecture

Physical Sensors
ESP32 polls medical & environmental sensors and packages data into JSON payloads.
Cloud Backend API
Node.js server processes incoming data, calculates BQI, and stores it in the time-series log.
Web Client
Browser pulls data via REST API to render dynamic charts and real-time alerts.
