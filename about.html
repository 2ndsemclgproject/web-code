BreathIQ Environmental Intelligence Platform

BreathIQ is an IoT-based environmental monitoring and health analytics platform developed to continuously assess air quality, environmental comfort, and physiological response in real time. The system combines multiple environmental sensors with cloud analytics to generate an intelligent Breath Quality Index (BQI) and Air Quality Index (AQI). Live sensor data is transmitted every 5 seconds, processed by a Node.js backend, analyzed using custom algorithms, and stored as one-minute averaged records inside MongoDB Atlas for long-term analytics.

Project Overview

Environmental Monitoring

BreathIQ continuously monitors indoor environmental conditions using gas sensing, dust sensing, temperature sensing, and humidity sensing technologies. The platform identifies pollution trends and assists in maintaining healthier indoor environments.

Health Monitoring

Physiological measurements such as blood oxygen saturation (SpO₂) is integrated into the system to evaluate the potential impact of environmental conditions on human wellbeing.

Analytics Engine

A dedicated backend processes incoming sensor data, calculates AQI, generates Breath Quality Index scores, classifies risk levels, and prepares data for dashboard visualization and historical analysis.

Sensor Suite

MQ135 Gas Sensor

The MQ135 sensor is responsible for detecting air contaminants and harmful gases present in indoor environments.

  • Smoke Detection
  • Ammonia Detection
  • Benzene Detection
  • VOC Monitoring
  • General Air Pollution Analysis

GP2Y10 Dust Sensor

The GP2Y10 optical dust sensor measures airborne particulate matter and suspended particles.

  • Dust Concentration Measurement
  • Particulate Matter Detection
  • AQI Calculation Input
  • Indoor Pollution Monitoring

DHT11 Sensor

The DHT11 provides temperature and humidity measurements which are used to evaluate environmental comfort.

  • Temperature Monitoring
  • Humidity Monitoring
  • Comfort Analysis
  • BQI Contribution

MAX30102 Pulse Oximeter

The MAX30102 measures blood oxygen saturation levels to evaluate physiological responses.

  • SpO₂ Measurement
  • Health Assessment
  • BQI Contribution

Breath Quality Index (BQI)

BreathIQ generates a proprietary Breath Quality Index (BQI) by combining multiple environmental and physiological factors.


Parameter Weight

Live Data & Historical Storage

Live Monitoring

Sensor readings are transmitted from the ESP32 to the backend every 5 seconds. The dashboard immediately reflects the latest environmental conditions.

Historical Database

To reduce database load, the backend collects multiple readings for one minute, computes averaged values, and stores a single optimized record in MongoDB Atlas.
